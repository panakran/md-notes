# Mark down notes app

## installing app

---

> clone repo https://github.com/panakran/md-notes.git

```cmd
git clone https://github.com/panakran/md-notes.git
```

> install app

```
npm install
```
> generate bundle

```
webpack
```

> run de-server

```
npm start
```
